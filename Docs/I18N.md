# KLineLens MVP - i18n / Language Settings

## 1) 目标
- 右上角 Settings 页面手动选择语言：中文/English
- 全站 UI 与报告文案随语言变化
- MVP 不做后端语言接口，优先前端字典渲染

## 2) 存储
- key: `kline_lens_lang`
- value: `zh` or `en`
- storage: localStorage（MVP）

## 3) 文案策略（关键）
- 后端返回尽量用 key（reason_key / note_key / if_key / risk_key）
- 前端根据 `lang` 映射到具体文本
- 避免后端直接输出一长段中文，未来加英文会很痛
